# donob
AcyOrt theme donob

## demo
http://acyort.am0200.com/

## Install
```bash
$ git clone git@github.com:acyortjs/theme-donob.git themes/donob
```

## Plugins and Config

Install the plugins

- acyort-rss
- acyort-archives

The config example

```yml
plugins:
  - acyort-rss
  - acyort-archives

rss:
  path: rss.xml

archives:
  per_page: 5
```
